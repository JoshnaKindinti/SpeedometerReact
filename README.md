# Speedometer Application

A simple and interactive **Speedometer** application built using React. This application allows users to increase or decrease the speed within a specified range (0 mph to 200 mph) using interactive buttons.

---

## 📝 Description

The Speedometer application is a classic state-management project built with React class components. It simulates a vehicle's speedometer dashboard. 

### Key Features
* **Accelerate:** Increases the speed by **10 mph** on every click, up to a maximum limit of **200 mph**.
* **Apply Brake:** Decreases the speed by **10 mph** on every click, down to a minimum limit of **0 mph**.
* **Boundary Constraints:** Prevents the speed from going below 0 or exceeding 200, ensuring safe operational logic.

---

## 🛠️ Tech Stack

* **Frontend:**React.js (Class Components)
* **Styling:** CSS3

---

## 🚀 How to Run the Application

Follow these steps to get your local development server up and running.

### 1. Prerequisites
Make sure you have **Node.js** and **npm** installed on your machine. You can check by running:
node -v
npm -v

---

### 2. Setup the Project
If you haven't already set up a React app, create one using Create React App:

npx create-react-app speedometer-app
cd speedometer-app
